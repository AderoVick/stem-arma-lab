# Changelog

## 2.1.0 — 2026-07-20

- Added a dedicated responsive GitHub Pages landing page.
- Added project SEO metadata, structured data, social preview, favicon, sitemap, and robots file.

- Rebuilt the Streamlit dashboard with Overview, Explore, Forecast, and Diagnostics sections.
- Added robust CSV validation and a data-quality report.

- Added multi-topic analysis, topic suggestions, and exact/stem matching modes.
- Added model backtesting, automatic model selection, naive baseline comparison, and metrics.
- Added confidence intervals, non-negative forecast handling, residual diagnostics, and exports.
- Expanded the sample dataset and automated tests.
- Added CI, Docker, Streamlit configuration, architecture documentation, and an MIT license.
